# RC Motogarage — Creative Reference Library

This folder structure is organized for the creative agent to find design references quickly.

## Folder Structure

```
/assets/references/
├── brand-assets/       # Original campaign visuals, logos, product shots (COPIED FROM ROOT)
├── inspiration/        # General design inspiration, moodboards, screenshots
├── competitors/        # Competitor sites, campaigns, UI patterns
├── color-palettes/     # Color schemes, gradients, palette exports
├── typography/         # Font pairings, type scales, typographic references
├── ui-patterns/        # Specific UI components (cards, navs, forms, buttons)
└── mockups/            # Design mockups, wireframes, prototypes
```

## Brand Guidelines (from live site)

**Colors:**
- Background: `#141414` (--bg)
- Surface: `#1E1E1E` (--bg-2)
- Elevated: `#2A2A2A` (--bg-3)
- Primary Red: `#ED1C24` (--red)
- Muted Text: `#B8B4AE` (--muted)
- White: `#FFFFFF`

**Typography:**
- Font: Montserrat (weights 400, 500, 600, 700, 800)
- All caps for labels/nav, sentence case for body

**Style:**
- Dark theme, red accent
- Film grain overlay
- Rounded cards (14-16px radius)
- Pill buttons (999px radius)
- Sticky nav with blur backdrop

## Usage for Creative Agent

When asked to design new pages/components:
1. Check `brand-assets/` for existing visual language
2. Check `inspiration/` for collected references
3. Check `ui-patterns/` for reusable component patterns
4. Maintain consistency with colors/typography above
